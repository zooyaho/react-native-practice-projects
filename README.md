# react-native-practice-projects

React Native로 만든 다양한 실습용 토이 프로젝트 모음입니다.  
각 프로젝트별로 특정 기능과 개념을 집중적으로 학습하고 적용해보는 데 초점을 맞췄습니다.

## 사용 방법

- 각 프로젝트 폴더로 이동 후,

```bash
npm install
expo start
```

- Expo Go 앱 또는 시뮬레이터에서 실행하세요.
